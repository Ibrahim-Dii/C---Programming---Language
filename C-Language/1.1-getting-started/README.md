# 1.1 Getting Started

This section introduces the basic structure of a C program with the classic "hello, world" example.

## Files:
- `hello.c`: Basic hello world program
- `hello_multiple.c`: Multiple printf statements version

## Key Concepts:
- `#include <stdio.h>`: Includes standard input/output library
- `main()`: Entry point of every C program
- `printf()`: Function to print output
- `\n`: Newline character

## Compilation and Execution:
```bash
cc hello.c -o hello
./hello
